# commands/

## Overview

Slash command definitions for the roammem plugin. Each `.md` file is a user-invocable command with frontmatter `description` and a prompt body.

## Structure

- `roam.md` — `/roammem:roam` activates roam-first mode
- `gen.md` — `/roammem:gen` populates skeleton notes from source
- `save.md` — `/roammem:save` distills session learnings into notes
- `rule.md` — `/roammem:rule` emits a CLAUDE.md rule snippet
