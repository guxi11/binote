# commands/

## Overview

Slash command definitions for the roammem plugin. Each `.md` file is a user-invocable command with frontmatter `description` and a prompt body.

## Structure

- `roam.md` — `/roam` activates roam-first mode
- `roam-gen.md` — `/roam-gen` populates skeleton notes from source
- `roam-save.md` — `/roam-save` distills session learnings into notes
- `roam-rule.md` — `/roam-rule` emits a CLAUDE.md rule snippet
